# NBA names corpus for the Hoops Spelling Bee (v3).
#
# Format: one entry per line, three pipe-delimited fields:
#     NAME|TYPE|DISPLAY
#   - NAME    uppercase, A–Z only, diacritics stripped, punctuation removed
#   - TYPE    one of: last, first_mononym, nickname
#   - DISPLAY original casing + punctuation, used in the found-names list
#
# Length rule: NAME between 4 and 10 letters. Names of valid players whose
# name length falls outside that window are intentionally excluded — the Bee
# can't form them on a 7-letter board anyway.
#
# Inclusion rule (v3 starter, hand-curated):
#   - Currently rostered players you'd recognize from a SportsCenter highlight
#   - All-time greats with broad cultural footprint
#   - Mononym first names that read as "iconic" (LeBron, Magic, Shaq, Wemby)
#   - Nicknames that have crossed into mainstream coverage (Greek Freak, etc.)
#
# This is intentionally a starter list, not exhaustive. Grow over time as we
# see what the Bee surfaces and what users complain about. v3.5+ will
# automate refresh from `nba_api` with a games-played fame floor.
